# FirePlayer
Mediaplayer for Amazon Firestick and HomeServer (github.com/uriegel/HomeServer) as HTTP Range Streamer
## Power saving when using Sony TV
TV url:
```http://<tv url>/sony```

PSK:

enter PSK in Sony TV settings and FireMusic settings
